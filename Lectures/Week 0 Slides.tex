\section{Why This Module?}
%----------------------------------------------------------------------------------------

\begin{frame}{The Industry Has Changed}
  This is \highlight{not} a course about speculating on cryptocurrency prices.
  
  \vspace{1em}
  \textbf{2024 was a turning point:}
  \begin{itemize}
    \item \textbf{January 2024}: US approves Bitcoin spot ETFs
    \item \textbf{May 2024}: US approves Ethereum spot ETFs
    \item \textbf{June 2024}: MiCA regulation enters force in EU
    \item BlackRock, Fidelity, Franklin Templeton now offer crypto products
    \item Total crypto market cap: $>$\$2 trillion
  \end{itemize}
  
  \vspace{1em}
  Blockchain has moved from the fringes to \highlight{mainstream financial infrastructure}.
  
  \vspace{0.5em}
  The question is no longer ``Will this matter?'' but ``How will it reshape finance?''
\end{frame}

\section{Module Overview}
%----------------------------------------------------------------------------------------

\begin{frame}{What is This Module About?}
  \textbf{Core objective:} Understand blockchain technology and digital assets from an \highlight{economic} perspective---not a technical one.
  
  \vspace{1em}
  \begin{columns}[T]
    \column{0.48\textwidth}
    \textbf{We will cover:}
    \begin{itemize}
      \item How blockchain works (conceptually)
      \item DeFi: lending, trading, stablecoins
      \item Tokenization of real assets
      \item Crypto as an investment
      \item Regulation (MiCA, US, global)
      \item Practical due diligence
    \end{itemize}
    
    \column{0.48\textwidth}
    \textbf{Key features:}
    \begin{itemize}
      \item No coding required
      \item Economic intuition focus
      \item Real case studies
      \item Current developments
      \item Critical perspective
      \item Employability skills
    \end{itemize}
  \end{columns}
  
  \vspace{1em}
  \textbf{Philosophy:} Balanced assessment---neither crypto-booster nor crypto-skeptic. Facts and frameworks, not opinions.
\end{frame}

  \vspace{0.5em}

\begin{frame}{Lecture Structure}
  Each \highlight{2-hour lecture} combines:
  
  \vspace{1em}
  \begin{columns}[T]
    \column{0.48\textwidth}
    \textbf{Conceptual Foundations:}
    \begin{itemize}
      \item How things work
      \item Economic mechanisms
      \item Trade-offs and limitations
      \item Key terminology
    \end{itemize}
    
    \column{0.48\textwidth}
    \textbf{Applications and Cases:}
    \begin{itemize}
      \item Real-world examples

    \item Multiple choice
    \item Short answer
    \item Scenario-based questions
  \end{itemize}
  
  \vspace{1em}
  \textbf{Coverage:}
  \begin{itemize}
    \item Quiz 1: Weeks 1--4 (Foundations, Consensus, Smart Contracts, DeFi)

  \end{itemize}

\begin{frame}{Group Assignment (20\%)}
  \textbf{Case study or research project} in small groups
  
  \vspace{1em}
  \textbf{Possible topics:}
  \begin{itemize}
    \item Deep dive on a DeFi protocol (mechanics, risks, governance)
    \item Analysis of a stablecoin's reserve structure and risks
    \item Evaluation of a tokenization project
    \item Regulatory comparison across jurisdictions
    \item Post-mortem of a failure (Terra, FTX, etc.)
  \end{itemize}
  
  \vspace{1em}
  \textbf{Deliverables:}
  \begin{itemize}
    \item Written report (2,500--3,000 words)
    \item Brief presentation to class
  \end{itemize}
  
  \vspace{1em}
  \textbf{Assessment criteria:} Depth of analysis, use of data, critical evaluation, clarity of communication.
\end{frame}

\begin{frame}{Final Exam (50\%)}
  \textbf{2-hour written exam} covering all course material
  
  \vspace{1em}
  \textbf{Question types:}
  \begin{itemize}
    \item \textbf{Conceptual questions:} Explain mechanisms, compare approaches
    \item \textbf{Applied scenarios:} Evaluate a project, identify risks, recommend actions
    \item \textbf{Case analysis:} Given information about a protocol/token, assess it
    \item \textbf{Policy questions:} Discuss regulatory trade-offs
  \end{itemize}
  
  \vspace{1em}
  \textbf{What we're testing:}
  \begin{itemize}
    \item Understanding of core concepts
    \item Ability to apply frameworks to new situations
    \item Critical thinking and balanced assessment
    \item Clear communication of complex ideas
  \end{itemize}
\end{frame}

\section{Course Philosophy}
%----------------------------------------------------------------------------------------

\begin{frame}{Our Approach: Critical but Fair}
  \textbf{What this course is NOT:}
  \begin{itemize}
    \item A crypto promotion or investment guide
    \item A dismissive critique of ``fake money''
    \item A technical programming course
    \item Coverage of every blockchain project
  \end{itemize}
  
  \vspace{1em}
  \textbf{What this course IS:}
  \begin{itemize}
    \item Rigorous economic analysis of blockchain technology
    \item Balanced assessment of opportunities \textit{and} risks
    \item Focus on understanding mechanisms, not predicting prices
    \item Preparation for careers that require blockchain literacy
  \end{itemize}
  
  \vspace{1em}
  \begin{block}{Guiding Principle}
    If it seems too good to be true, it probably is. Our job is to understand \textit{why}---and to identify what \textit{does} have genuine value.
  \end{block}
\end{frame}

\begin{frame}{Distinguishing Hype from Substance}
  Throughout the course, we will develop frameworks to evaluate claims:
  
  \vspace{1em}
  \begin{columns}[T]
    \column{0.48\textwidth}
    \textbf{Questions to ask:}
    \begin{itemize}
      \item What problem does this solve?
      \item Why does it need a blockchain?
      \item What are the trust assumptions?
      \item Who benefits and who bears risk?
      \item What's the regulatory status?
      \item What could go wrong?
    \end{itemize}
    
    \column{0.48\textwidth}
    \textbf{Red flags:}
    \begin{itemize}
      \item Guaranteed returns
      \item ``Decentralised'' with admin keys
      \item Complexity without purpose
      \item Unaudited smart contracts
      \item Anonymous teams
      \item Regulatory arbitrage
    \end{itemize}
  \end{columns}
  
  \vspace{1em}
  By Week 11, you will have a systematic framework for this analysis.
\end{frame}
